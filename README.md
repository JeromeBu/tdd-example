You need node 12+ to use the app :

**Install**

```
npm install
```

**To test the app :**

```
npm run test:all
```

You can run in watch mode individually :
Unit tests :

```
npm run test:unit
```

Integration tests :

```
npm run test:integration
```

End to end tests :

```
npm run test:e2e
```

**Start the app :**

```
npm run start
```
